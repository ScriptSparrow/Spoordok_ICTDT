Hier gaan we dingen neerzetten zodat iedereen makkelijk weer kan opstarten.

## Excalidraw 

Voor de tekeningen gebruik ik Excalidraw. <br>
Een programma waar makkelijk schematisch in te schetsen is. 

Voor zowel Intellij als VS Code zijn er plugins die het direct in de IDE mogelijk maken te editen. 

- [Excalidraw Extension VS Code](https://marketplace.visualstudio.com/items?itemName=pomdtr.excalidraw-editor) - Klik op "Install" om direct te openen in VS Code
- [Excalidraw Integration IntelliJ](https://plugins.jetbrains.com/plugin/17096-excalidraw-integration) - Klik op "Install to IDE" om direct te openen in IntelliJ
